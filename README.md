# ml-journey

A public log of my transition from senior full-stack engineering into ML research. Started June 2026.

I'm Gregory Kemp. Background: BEng Mechanical Engineering, MSc Fusion Energy, 11+ years building production software. The goal is a research engineering role at a frontier lab (DeepMind, Anthropic, or equivalent tier) by 2029.

This repo is the working artifact: code, paper reproductions, and notes. The writing that goes with it lives on my site at [kempgt.com/blog](https://kempgt.com/blog) — each post links back to the relevant code here. Everything is built in public, both because hiring managers read code rather than certificates, and because writing in public is the fastest way to find out what I don't actually understand.

## Current focus

- Karpathy's *Neural Networks: Zero to Hero*, starting with `micrograd`
- Reading: Bishop PRML Ch. 1
